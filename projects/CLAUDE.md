## Quick Route

When you're not sure where to start, match the question shape:

- **"What does this ecosystem mean by X?"** → `core/language-notes.md`
- **"How should AI and human collaborate here?"** → `frameworks/meta-protocol.md`
- **"Is AI misreading this cultural pattern?"** → `frameworks/ai-cultural-literacy.md`
- **"How do I safely integrate an external tool?"** → `frameworks/decon.md`
- **"Is the system healthy?"** → `frameworks/engine-protection.json`
- **"What's the ethical foundation?"** → `frameworks/negentropic-principle.md`
- **"How does [physical system] work?"** → `resilience/` (match cluster above)
- **"What's the physics behind X?"** → `experiments/` (match cluster above)
- **"How do emotions/dreams/sensing work here?"** → `vault/`
- **"Where did that idea go?"** → `field-notes/wander.md`
- **"What's pending?"** → `core/to-do-cache.md`
- **"How do I teach this in 15 minutes?"** → `heritage/micro-modules.md`
- **"Emergency?"** → `heritage/emergency-playbook.txt`
- **"How does the filter/detection work?"** → `prototypes/`
- **"Where does everything connect?"** → `core/CORE.json`
- **"What drives this whole project?"** → `heritage/manifesto.txt`

**Cross-domain questions** — If a question touches multiple folders, check
`core/CORE.json` for the lattice connections, then visit each relevant folder.
The folders are ecological — boundaries are porous by design.

**New ideas with no home** — Start in `field-notes/`. Cross-reference in
`field-notes/wander.md` or `core/to-do-cache.md`. It will migrate when ready.
