## Adding a new repo

1. Start with the universal set.
2. Pick up to 23 repo-specific topics (universal has 7 = 30 budget).
3. Prefer existing GitHub topics over inventing new ones — corpus
   classifiers more readily index canonical tags.
4. If inventing a new tag, document it here for cross-repo consistency.
